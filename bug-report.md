Bug Report – Aplicação Web

BUG-001 – Sistema permite login sem validação
Severidade: Alta
Prioridade: Alta
Ambiente: Web – Google Chrome

Descrição:
O sistema permite efetuar login sem validação adequada dos campos obrigatórios.

Passos para reprodução:
1. Acessar a tela de login
2. Deixar os campos de usuário e senha em branco
3. Clicar em “Entrar”

Resultado esperado:
Sistema deve bloquear o login e exibir mensagens de validação

Resultado obtido:
Sistema permite tentativa de login sem validação
